Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential>
class cattle::SoftmaxCrossEntropyLoss< Scalar, Rank, Sequential >

A loss function template that applies the softmax function to its input before calculating the cross entropy loss. This allows for increased numerical stability and faster computation.
