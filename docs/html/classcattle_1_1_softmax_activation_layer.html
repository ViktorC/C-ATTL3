Detailed Description

template<typename Scalar, std::size_t Rank>
class cattle::SoftmaxActivationLayer< Scalar, Rank >

A class template for a softmax activation function layer. Unlike most other activation layers which represent element-wise functions, the softmax layer represents a multivariate function.
