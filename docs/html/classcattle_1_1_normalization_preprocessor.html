template<typename Scalar, std::size_t Rank, bool Standardize = false>
class cattle::NormalizationPreprocessor< Scalar, Rank, Standardize >

A class template for a normalization (mean-subtraction) preprocessor that optionally also standardizes the data (divides it by the standard deviation).
