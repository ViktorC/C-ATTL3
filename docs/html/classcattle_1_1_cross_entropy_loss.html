Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential>
class cattle::CrossEntropyLoss< Scalar, Rank, Sequential >

A template class representing the cross entropy loss function. This class assumes the objective values for each sample (and time step) to be in the range [0, 1].
