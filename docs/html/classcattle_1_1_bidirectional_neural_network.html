Detailed Description

template<typename Scalar, std::size_t Rank>
class cattle::BidirectionalNeuralNetwork< Scalar, Rank >

A class template for a bidirectional neural network that takes a unidirectional recurrent network, clones it, reverses the clone's processing direction, and uses the two networks as its parallel sub-modules. The outputs of the two sub-networks can be merged by summation or concatenation either along the lowest (the 3rd after the sample and time-step ranks) or highest rank.
