Detailed Description

template<typename Scalar>
class cattle::DeconvKernelLayer< Scalar, 2 >

A class template for a transposed 2D convolutional layer operating on rank-2 data batches (rank-3 tensors).

The results of the convolutions of the filters and the input tensor are concatenated along the highest (3rd) rank of the output tensor.

See also
https://arxiv.org/abs/1603.07285v1

input_dimsThe dimensionality of the observations to be processed by the layer. The ranks of the input tensors denote the sample, height, and width (N,H,W).
filtersThe number of filters to use.
weight_initA shared pointer to a weight initialization used to initialize the values of the parametric kernel backing the layer.
weight_regThe regularization function to apply to the layer's parameters.
receptor_heightThe height of the receptor field.
receptor_widthThe width of the receptor field.
vertical_paddingThe extent of padding to apply to the input tensor along its height (both at the top and at the bottom).
horizontal_paddingThe extent of padding to apply to the input tensor along its width (both at the left and at the right).
vertical_strideThe vertical convolution stride i.e. the number of elements by which the receptor is to be shifted along the height of the input tensor.
horizontal_strideThe horizonzal convolution stride i.e. the number of elements by which the receptor is to be shifted along the width of the input tensor.
vertical_dilationThe extent of vertical dilation to apply to the receptor.
horizontal_dilationThe extent of horizontal dilation to apply to the receptor.
max_norm_constraintAn optional max-norm constraint. If it is 0 or less, no constraint is applied.
