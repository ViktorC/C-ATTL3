Detailed Description

template<typename Derived, typename IndexType, std::size_t Rank>
class cattle::DimExpression< Derived, IndexType, Rank >

NOTE: Expression templates are rather unwarranted for this class as its objects are unlikely to be used in complex expressions.

The base class of all dimension expressions.

They have only been implemented for the sake of practical learning.
