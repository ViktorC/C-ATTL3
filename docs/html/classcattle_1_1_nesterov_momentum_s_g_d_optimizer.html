Parameters
lossA shared pointer to the loss function to use.
batch_sizeThe batch size to use for training and testing. It is expected to be greater than 0.
init_learning_rateThe initial learning rate (a.k.a. step size) to use. It is expected to be greater than 0.
annealing_rateThe rate at which the learning rate is to be annealed. It is expected to be greater than or equal to 0. The greater it is, the faster the learning rate decreases.
momentumThe momentum rate to use. The greater the momentum, the lesser the effect of newer gradients. It is expected to be greater than 0 and less than 1.
