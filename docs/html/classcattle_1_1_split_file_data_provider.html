Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential>
class cattle::SplitFileDataProvider< Scalar, Rank, Sequential >

An abstract class template for a data provider backed by an arbitrary number of file pairs containing the separated observations and the objectives. Implementations are responsible for specifying the dimensions of both the observations and the objectives, for reading batches of observation-objective pairs from the file, and for skipping arbitrary number of data instances.
