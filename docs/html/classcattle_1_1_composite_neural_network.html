Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential, typename Module>
class cattle::CompositeNeuralNetwork< Scalar, Rank, Sequential, Module >

A class template for composite neural networks consisting of one or more neural network modules.

virtual std::vector<Module*> cattle::CompositeNeuralNetwork< Scalar, Rank, Sequential, Module >::get_modules ( )
pure virtual
Returns
A vector of pointers pointing to the sub-modules of the composite network instance. The ownership of the modules is not transferred to the caller of the method.
