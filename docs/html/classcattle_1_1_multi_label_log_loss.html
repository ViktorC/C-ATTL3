Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential>
class cattle::MultiLabelLogLoss< Scalar, Rank, Sequential >

A class template representing the logarithmic loss function for multi-label objectives. True labels are expected to have the value 1, while false labels are expected to correspond to the value 0.
