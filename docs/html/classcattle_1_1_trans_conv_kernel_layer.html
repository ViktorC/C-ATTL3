Detailed Description

template<typename Scalar, std::size_t Rank = 3>
class cattle::TransConvKernelLayer< Scalar, Rank >

A class template for a transposed 2D convolutional layer operating on rank-3 data batches (rank-4 tensors).

The results of the convolutions of the filters and the input tensor are concatenated along the highest (4th) rank of the output tensor.

See also
https://arxiv.org/abs/1603.07285v1
