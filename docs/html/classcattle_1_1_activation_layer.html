◆ is_input_layer()

template<typename Scalar, std::size_t Rank>
bool cattle::ActivationLayer< Scalar, Rank >::is_input_layer ( ) const
inlineprotectedvirtual

A constant method that returns whether this layer functions as an input layer. An input layer does not need to propagate the gradients all the way during the backward pass as it is assumed that no other layer needs them derive the gradient on its parameters. It is therefore possible for an input layer to simply return a null tensor as the output of its backward pass.

Returns
Whether this layer is the input layer of the neural network that contains it.
