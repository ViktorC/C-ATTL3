Detailed Description

template<typename Scalar, std::size_t Rank, bool MulInt = false, bool Stateful = false>
class cattle::RecurrentNeuralNetwork< Scalar, Rank, MulInt, Stateful >

A class template for a simple recurrent neural network (RNN).

The network can use multiplicative integration to combine its linearly transformed input and its linearly transformed previous hidden state. A stateful network retains its hidden state across sequences as long as the batch size is constant.
