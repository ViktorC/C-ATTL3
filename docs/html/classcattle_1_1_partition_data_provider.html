Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential>
class cattle::PartitionDataProvider< Scalar, Rank, Sequential >

A wrapper class template for data providers associated with continuous partitions of other data providers. It enables the partitioning of a data provider into training and test data providers by mapping two contiguous blocks of its data to two PartitionedDataProvider instances.
