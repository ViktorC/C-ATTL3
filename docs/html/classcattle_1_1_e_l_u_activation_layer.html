Detailed Description

template<typename Scalar, std::size_t Rank>
class cattle::ELUActivationLayer< Scalar, Rank >

A class template representing an exponential linear unit (ELU) activation function. ELUs apply an exponential (e based) function scaled by alpha to the negative elements of the input. ELU layers are not theoretically differentiable.
