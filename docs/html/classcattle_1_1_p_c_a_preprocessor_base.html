Detailed Description

template<typename Scalar, std::size_t Rank, bool Standardize = false, bool Whiten = false>
class cattle::PCAPreprocessorBase< Scalar, Rank, Standardize, Whiten >

An abstract base class template for a principal component analysis (PCA) preprocessor that can also standardize and whiten the data.
