It reads at most the specified number of observations from the observation-file and at most the specified number of objectives from the objective-file. The file streams can be expected not to have any of their fail flags set initially and to have at least 1 more character left to read in each.

Parameters
obs_input_streamA reference to the file stream to a file containing observations.
obj_input_streamA reference to the file stream to a file containing objectives.
batch_sizeThe number of data points to read.
Returns
