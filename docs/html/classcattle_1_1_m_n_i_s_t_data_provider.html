It reads at most the specified number of observations from the observation-file and at most the specified number of objectives from the objective-file.

The file streams can be expected not to have any of their fail flags set initially and to have at least 1 more character left to read in each.

Parameters
obs_fileThe name of the observation source file.
obs_file_streamThe input stream of the observation file.
obj_fileThe name of the objective source file.
obj_file_streamThe input stream of the objective file.
batch_sizeThe number of data points to read.
Returns
The paired observations and objectives.
