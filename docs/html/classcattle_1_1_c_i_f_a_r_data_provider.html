It reads at most the specified number of observation-objective pairs from the provided file stream.

The file stream can be expected not to have any of its fail flags set initially and to have at least 1 more character left to read.

Parameters
data_streamA reference to the file stream of the data set.
batch_sizeThe number of data points to return.
Returns
