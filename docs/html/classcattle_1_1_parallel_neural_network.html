Detailed Description

template<typename Scalar, std::size_t Rank>
class cattle::ParallelNeuralNetwork< Scalar, Rank >

A class template for a parallel neural network that consists of one or more lanes of non-sequential neural networks with the same input dimensions. Inputs and gradients are propagated through the lanes simultaneously using multithreading. The outputs of the lanes are merged by concatenation (either along the lowest or hightest rank) or summation.
