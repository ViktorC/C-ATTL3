Parameters
min_rel_var_to_retainThe minimum relative variance in the data to retain. It is expected to be within the range (0,1]. If it is 1, the dimensionality of the preprocessed data is guaranteed not to be reduced. If it is less than 1, the data cannot be a multi-channel tensor.
epsilonA small consant used to maintain numerical stability.
