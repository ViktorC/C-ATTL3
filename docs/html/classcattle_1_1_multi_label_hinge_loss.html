Detailed Description

template<typename Scalar, std::size_t Rank, bool Sequential, bool Squared = false>
class cattle::MultiLabelHingeLoss< Scalar, Rank, Sequential, Squared >

A class template representing the hinge loss function for multi-label objectives. True labels are expected to have the value 1, while false labels are expected to correspond to the value -1.
