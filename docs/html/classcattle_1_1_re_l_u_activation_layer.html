Detailed Description

template<typename Scalar, std::size_t Rank>
class cattle::ReLUActivationLayer< Scalar, Rank >

A class template representing a rectified linear unit (ReLU) activation function.

ReLU layers set all negative elements of the input to 0. This function is not differentiable.

\[ f(x) = \begin{cases} 0 & \text{for } x < 0\\ x & \text{for } x \geq 0 \end{cases} \]
